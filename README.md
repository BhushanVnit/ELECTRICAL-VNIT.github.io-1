# How to Use.

1. Download the [template](https://github.com/ABD-01/Academic-Time-Table/tree/main/template) folder.

2. Add the Subject Names in `subjects.js` file.
	```javascript
	"slot#": {
	    name: "Subject Name",
	    link: "Lecture Link"
	  }
	```
3. Done.
